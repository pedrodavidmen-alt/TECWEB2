Paso 1 de 6

El usuario ingresa una URL en el navegador

Todo comienza cuando escribes una dirección web, como https://www.ejemplo.com, en la barra de tu navegador. El navegador necesita traducir ese nombre de dominio humano en una dirección IP numérica mediante el sistema DNS (Domain Name System).

Dato clave: el DNS funciona como una agenda telefónica de internet: convierte nombres fáciles de recordar en direcciones IP que las máquinas pueden usar.
